# syntanaliz
Программа является синтаксическим анализатором. Проверяет правильность использования скобок (в том числе фигурных), а также использования break'ов. Подсвечены будут только те, которые привели бы к ошибке компиляции кода.
В Menu.cpp находится код небольшого меню в консоли.
В MainClass.cpp - реализация класса MainClass.
В MainClass.h - заголовое класса MainClass
